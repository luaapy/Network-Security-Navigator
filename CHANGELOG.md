# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-05
### Added
- Initial project scaffolding and architecture.
- Core virtual machine and object system skeletons.
- Lexer and recursive descent parser foundations.
- Standard library modules (network, security, crypto, utils).
- Build system support (CMake, Autotools, NDK).
- Packaging support (Docker, Debian, RPM, Arch, Homebrew, Flatpak, Snap).
- Documentation structure and initial tutorials.
