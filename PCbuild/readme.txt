NSN MSVC Build System
=====================

This directory contains Microsoft Visual Studio project files.

Projects:
- `nsn.sln`: Main solution.
- `nsn.vcxproj`: Core interpreter project.
- `build.bat`: Command-line script to initiate the build.

Requirements:
- Visual Studio 2022 or newer.
- Windows SDK 10.0.19041.0+.
- OpenSSL (set OPENSSL_DIR environment variable).
