# NSN Development Roadmap

## Phase 1: Prototype (Completed)
- Directory structure and scaffolding.
- Basic Lexer and Parser logic.
- Minimal VM and object system.
- Standard Library skeletons.

## Phase 2: Core Stabilization
- Full PEG parser implementation.
- Refining Bytecode compiler.
- Expanding network object support (IPv6, custom raw frames).
- Comprehensive unit test suite.

## Phase 3: Performance & Security
- Bytecode optimization.
- Secure sandbox execution mode.
- Advanced vulnerability auditing modules.
- SSL/TLS deep inspection support.

## Phase 4: Ecosystem & Tooling
- VS Code extension for NSN.
- Browser-based playground (WASM).
- Official package manager for NSN modules.
